# useForm

## Purpose

Manage form values, validation, and submit state.

## Import

~~~tsx
import { useForm } from "@tesseract-nexus/tesserix-ui"
~~~

## Example

~~~tsx
const form = useForm({
  initialValues: { email: "" },
  validate: (values) => (values.email ? {} : { email: "Email is required" }),
})
~~~
