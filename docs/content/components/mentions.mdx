# Mentions

## Overview

Textarea with `@mention` suggestions.

## Import

~~~tsx
import { Mentions } from "@tesseract-nexus/tesserix-ui"
~~~

## Usage

~~~tsx
<Mentions options={[{ id: "1", label: "mahesh" }]} />
~~~

## Do / Don't

### Do

~~~tsx
<Mentions className="w-full" />
~~~

### Don't

~~~tsx
<Mentions className="bg-[#123456] text-[#fafafa] px-[13px]" />
~~~
