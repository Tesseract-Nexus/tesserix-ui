# Cron Builder

## Overview

Cron Builder helps compose recurring hourly or weekly schedules and previews the cron expression.

## Import

~~~tsx
import { CronBuilder } from "@tesseract-nexus/tesserix-ui"
~~~

## Exports

~~~ts
export { CronBuilder } from "./cron-builder"
export type { CronBuilderProps, CronBuilderValue } from "./cron-builder"
~~~

## Usage

~~~tsx
<CronBuilder onValueChange={(value) => console.log(value)} />
~~~
