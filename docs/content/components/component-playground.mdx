# Component Playground

## Overview

Live code/input area with side-by-side preview surface.

## Import

~~~tsx
import { ComponentPlayground } from "@tesseract-nexus/tesserix-ui"
~~~

## Usage

~~~tsx
<ComponentPlayground preview={<Button>Preview</Button>} />
~~~

## Do / Don't

### Do

~~~tsx
<ComponentPlayground className="w-full" />
~~~

### Don't

~~~tsx
<ComponentPlayground className="bg-[#123456] text-[#fafafa] px-[13px]" />
~~~
