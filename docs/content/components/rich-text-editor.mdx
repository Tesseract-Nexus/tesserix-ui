# Rich Text Editor

## Overview

WYSIWYG editor with basic formatting controls.

## Import

~~~tsx
import { RichTextEditor } from "@tesseract-nexus/tesserix-ui"
~~~

## Usage

~~~tsx
<RichTextEditor onValueChange={setHtml} />
~~~

## Do / Don't

### Do

~~~tsx
<RichTextEditor className="w-full" />
~~~

### Don't

~~~tsx
<RichTextEditor className="bg-[#123456] text-[#fafafa] px-[13px]" />
~~~
