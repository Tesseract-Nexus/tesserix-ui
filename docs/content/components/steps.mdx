# Usteps

## Overview

Usteps is part of the Tesserix UI component library.

## Import

~~~tsx
import { Usteps } from "@tesseract-nexus/tesserix-ui"
~~~

## Basic Usage

~~~tsx
import { Usteps } from "@tesseract-nexus/tesserix-ui"

export function Example() {
  return <Usteps />
}
~~~

## Notes

- Compose this primitive with domain-specific wrappers in app code.
- Prefer design tokens and existing variants before adding new styles.

## Do / Don't

### Do

~~~tsx
<Steps className="w-full" />
~~~

### Don't

~~~tsx

~~~
