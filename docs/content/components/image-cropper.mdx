# Image Cropper

## Overview

Basic square cropper with zoom and export callback.

## Import

~~~tsx
import { ImageCropper } from "@tesseract-nexus/tesserix-ui"
~~~

## Usage

~~~tsx
<ImageCropper src={imageUrl} onCrop={setCroppedDataUrl} />
~~~

## Do / Don't

### Do

~~~tsx
<ImageCropper className="w-full" />
~~~

### Don't

~~~tsx
<ImageCropper className="bg-[#123456] text-[#fafafa] px-[13px]" />
~~~
