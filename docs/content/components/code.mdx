# Ucode

## Overview

Ucode is part of the Tesserix UI component library.

## Import

~~~tsx
import { Ucode } from "@tesseract-nexus/tesserix-ui"
~~~

## Basic Usage

~~~tsx
import { Ucode } from "@tesseract-nexus/tesserix-ui"

export function Example() {
  return <Ucode />
}
~~~

## Notes

- Compose this primitive with domain-specific wrappers in app code.
- Prefer design tokens and existing variants before adding new styles.

## Do / Don't

### Do

~~~tsx
<Code className="w-full" />
~~~

### Don't

~~~tsx
<Code className="bg-[#123456] text-[#fafafa] px-[13px]" />
~~~
