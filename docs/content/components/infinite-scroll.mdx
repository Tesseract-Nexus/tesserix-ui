# Uinfinite Uscroll

## Overview

Uinfinite Uscroll is part of the Tesserix UI component library.

## Import

~~~tsx
import { UinfiniteUscroll } from "@tesseract-nexus/tesserix-ui"
~~~

## Basic Usage

~~~tsx
import { UinfiniteUscroll } from "@tesseract-nexus/tesserix-ui"

export function Example() {
  return <UinfiniteUscroll />
}
~~~

## Notes

- Prefer design tokens and existing variants before adding new styles.
