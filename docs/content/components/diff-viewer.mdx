# Diff Viewer

## Overview

Diff Viewer compares previous and next text content in split or unified layouts.

## Import

~~~tsx
import { DiffViewer } from "@tesseract-nexus/tesserix-ui"
~~~

## Exports

~~~ts
export { DiffViewer } from "./diff-viewer"
export type { DiffViewerProps } from "./diff-viewer"
~~~

## Usage

~~~tsx
<DiffViewer oldValue={beforeText} newValue={afterText} view="split" />
~~~

## Do / Don't

### Do

~~~tsx
<DiffViewer className="w-full" />
~~~

### Don't

~~~tsx
<DiffViewer className="bg-[#123456] text-[#fafafa] px-[13px]" />
~~~
