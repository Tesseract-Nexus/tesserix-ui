# Markdown Editor

## Overview

Side-by-side markdown input and live preview.

## Import

~~~tsx
import { MarkdownEditor } from "@tesseract-nexus/tesserix-ui"
~~~

## Usage

~~~tsx
<MarkdownEditor value={markdown} onValueChange={setMarkdown} />
~~~

## Do / Don't

### Do

~~~tsx
<MarkdownEditor className="w-full" />
~~~

### Don't

~~~tsx
<MarkdownEditor className="bg-[#123456] text-[#fafafa] px-[13px]" />
~~~
