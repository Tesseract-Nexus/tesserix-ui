# Uicon Ulibrary

## Overview

Uicon Ulibrary is part of the Tesserix UI component library.

## Import

~~~tsx

~~~

## Basic Usage

~~~tsx
import { UiconUlibrary } from "@tesseract-nexus/tesserix-ui"

export function Example() {
  return <UiconUlibrary />
}
~~~
