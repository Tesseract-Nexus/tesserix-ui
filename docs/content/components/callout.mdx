# Ucallout

## Overview

Ucallout is part of the Tesserix UI component library.

## Import

~~~tsx
import { Ucallout } from "@tesseract-nexus/tesserix-ui"
~~~

## Basic Usage

~~~tsx
import { Ucallout } from "@tesseract-nexus/tesserix-ui"

export function Example() {
  return <Ucallout />
}
~~~

## Notes

- Compose this primitive with domain-specific wrappers in app code.
- Prefer design tokens and existing variants before adding new styles.

## Do / Don't

### Do

~~~tsx
<Callout className="w-full" />
~~~

### Don't

~~~tsx
<Callout className="bg-[#123456] text-[#fafafa] px-[13px]" />
~~~
