# Charts

## Overview

Basic chart primitives for dashboards: bar and line.

## Import

~~~tsx
import { BarChart, LineChart } from "@tesseract-nexus/tesserix-ui"
~~~

## Usage

~~~tsx
<BarChart data={series} />
<LineChart data={series} />
~~~

## Do / Don't

### Do

~~~tsx
<Charts className="w-full" />
~~~

### Don't

~~~tsx
<Charts className="bg-[#123456] text-[#fafafa] px-[13px]" />
~~~
