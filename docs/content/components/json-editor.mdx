# Json Editor

## Overview

Json Editor provides a JSON textarea with validation plus format and minify actions.

## Import

~~~tsx
import { JsonEditor } from "@tesseract-nexus/tesserix-ui"
~~~

## Exports

~~~ts
export { JsonEditor } from "./json-editor"
export type { JsonEditorProps } from "./json-editor"
~~~

## Usage

~~~tsx
<JsonEditor value={{ enabled: true }} onChange={(value) => console.log(value)} />
~~~
