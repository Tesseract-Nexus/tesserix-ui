# Tour

## Overview

Guided onboarding flow rendered as a modal overlay with step navigation.

## Import

~~~tsx
import { Tour } from "@tesseract-nexus/tesserix-ui"
~~~

## Usage

~~~tsx
<Tour open={open} onOpenChange={setOpen} steps={steps} />
~~~

## Do / Don't

### Do

~~~tsx
<Tour className="w-full" />
~~~

### Don't

~~~tsx
<Tour className="bg-[#123456] text-[#fafafa] px-[13px]" />
~~~
