# Kanban

## Overview

Simple kanban board with column-based card movement.

## Import

~~~tsx
import { Kanban } from "@tesseract-nexus/tesserix-ui"
~~~

## Usage

~~~tsx
<Kanban columns={columns} />
~~~

## Do / Don't

### Do

~~~tsx
<Kanban className="w-full" />
~~~

### Don't

~~~tsx
<Kanban className="bg-[#123456] text-[#fafafa] px-[13px]" />
~~~
