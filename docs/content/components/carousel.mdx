# Ucarousel

## Overview

Ucarousel is part of the Tesserix UI component library.

## Import

~~~tsx

~~~

## Basic Usage

~~~tsx
import { Ucarousel } from "@tesseract-nexus/tesserix-ui"

export function Example() {
  return <Ucarousel />
}
~~~
